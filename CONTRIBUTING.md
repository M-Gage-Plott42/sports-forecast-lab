# Contributing

This export accepts only research-only, public-source changes.

Before proposing changes:

1. Keep private ledgers, app/slate records, screenshots, account details,
   transaction details, model binaries, generated bulk artifacts, and secrets
   out of the tree.
2. Preserve source/access-status fields for any public source row.
3. Add or update model cards and dataset cards for reviewed research artifacts.
4. Run the public validators.

Current sport-adapter work is fixture-only MLB scaffolding. Do not add model
training or live recommendation workflows until source rows and validation
contracts are stable and explicitly approved.
