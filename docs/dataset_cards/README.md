# Dataset Cards

Dataset cards document public research datasets before they are used as more
than exploratory context.

Each reviewed dataset card should include:

- intended research use;
- prohibited use;
- source/access status;
- collection and refresh method;
- schema and row counts;
- exclusions;
- leakage risks;
- maintenance plan;
- privacy/export boundary.

Dataset cards are research governance. They are not live-event or financial
instructions.
